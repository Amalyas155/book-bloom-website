## About Book Bloom
Book Bloom is a responsive website that helps readers discover books from different categories. It provides concise summaries and highlights the key learning outcomes of each book, making it easier for users to understand a book's value before deciding to read or purchase it.

##  Project Overview

Book Bloom was developed to simplify the process of discovering books through a clean and organized interface. Instead of relying only on book titles or covers, users can browse books by category, read a brief summary, and explore the key knowledge and skills each book offers.

The website aims to make book selection easier, faster, and more enjoyable for readers interested in different fields.

---

##  Features

- Responsive and modern website design
- Organized book categories
- Expandable category sections
- Book cover preview

##  How to Run

1. Clone or download this repository.
2. Make sure all image files are located in the same folder as `index.html`.
3. Open `index.html` using any modern web browser.
